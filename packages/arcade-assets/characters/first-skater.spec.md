# First Skater Spec: Rook Rocket

`rook-rocket` is an original agile skater for the first production pipeline pass.

## Visual Direction

- Oversized rounded helmet with a small rear fin.
- Compact torso with broad shoulder pads.
- Bright lace accents on skates and gloves.
- Team uniform slots carry jersey, pants, socks, numbers, and trim.
- Character gear slots own helmet, gloves, skates, and stick.

## Gameplay Read

Rook should read as fast and clean from the high camera: narrow stance, quick turn lean, and a crisp wrist-shot silhouette. The design must not reference a real player, NHL uniform, or real team mark.

- id: `rook-rocket`
