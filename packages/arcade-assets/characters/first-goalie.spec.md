# First Goalie Spec: Mira Wall

`mira-wall` is an original goalie for the first production pipeline pass.

## Visual Direction

- Wide squared pads with simple diagonal embossing.
- Smooth mask with a raised brow ridge.
- Oversized blocker and glove for clear save reads.
- Team uniform slots carry jersey, pants, socks, numbers, and trim.
- Character gear slots own mask, pads, gloves, skates, and stick.

## Gameplay Read

Mira should read as stable and defensive from the high camera: broad ready stance, lateral slide, pad save, and cover poses. The design must not reference a real goalie, NHL uniform, or real team mark.

- id: `mira-wall`

- required rig: all common bones and attachments from `rig-contract.md`
